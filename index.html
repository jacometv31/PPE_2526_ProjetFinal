Présentation du projet


Ce site a été créé dans le cadre du projet de programmation encadré du premier semestre de Master 1 Traitement Automatique des Langues et représente la synthèse de notre travail. Il vise à présenter le résultat de notre travail de collecte à partir d'un corpus trilingue tiré de pages web et étudier le comportement d'un mot de notre choix dans ces trois corpus. Les trois langues étudiées dans notre corpus sont le français, l'arabe et le japonais. Le mot que nous avons choisi est le mot espoir.


Pourquoi le mot espoir ?


Nous voulions choisir un mots désigant un sentiment puissant. Nous nous étions d'abord mis d'accord sur le mot Nostalgie. Toutefois, il avait déjà été utilisé pour un précédent projet. Après avoir bien réfléchis, nous avons décidé de nous tourner vers le mot Espoir, qui a plus de polysémie que notre précédent mot. Dans le sens le plus courrant, il s'agit d'un sentiment intérieur, qui est l’attente confiante qu’un événement positif se produise. Et dans un sens plus concret, il s'agit de ce que l'on attend, ce mot peut désigner la chose ou la personne qui fait espérer.

Contenu des tableaux


Nous avons d'abord rassemblé de nombreuses urls afin de pouvoir créer ces tableaux.

Les tableaux ont au minimum les catégories suivantes pour chaque langues:

  • - Le numéro de ligne
  • - Le code HTTP
  • - L'url
  • - L'encodage
  • - Le nombre d'occurrences du mot
  • - Le lien vers l'HTML brut
  • - Le lien vers le DUMP textuel
  • - Le lien vers le concordancier HTML
